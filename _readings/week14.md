#### Wiring

| Sensor | Argon                   |
| ------ | ----------------------- |
| 3.3v   | 3.3v                    |
| Reset  | A5 (any GPIO pin works) |
| GND    | GND                     |
| IRQ    | -                       |
| MOSI   | MO                      |
| MISO   | MI                      |
| SCK    | SCK                     |
| SDA    | A4 (any GPIO pin works) |
